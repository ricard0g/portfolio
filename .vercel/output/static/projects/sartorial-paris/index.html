Sartorial Paris: Building a Better Online Store

    → GitHub Repo
  • → Sartorial Store Live - Password: shorto

Sartorial Paris is a fully working online store I built from scratch to show what's possible with modern web development.

"A place where men can find well-made clothes that combine classic Parisian style with modern touches."

I got my inspiration from Todd Snyder's website, but I added my own ideas to make shopping easier and more enjoyable.

What I Set Out to Do

While Todd Snyder's website was my starting point, I wanted to make something that felt fresh and solved common shopping frustrations.

I focused on making the site fast and easy to use 🎯 by carefully choosing the right tools for the job.

What Makes This Store Special

🔍 Better Navigation

  • Menu categories that show options when you hover
  • Smart search that predicts what you're looking for by implementing the Predictive Search API from scratch
  • Navbar that follows you as you scroll
  • Mobile menu that's actually easy to use

🎯 Smooth Product Browsing

  • Touch-friendly product sliders
  • Quick-view product details without page loads
  • Continuous shopping without interruptions

🔎 Finding What You Want

  • Filters that understand what you're looking for through a full custom implementation of Shopify's Storefront Filtering where the UX guidelines were the highest priority
  • Smart sorting by size, color, and other features
  • Tag-based filtering that works on any device

👔 Helpful Suggestions

  • "You might also like" based on what you're viewing throught the Product Recommendations API
  • Fresh content that updates as you browse thanks to Section Rendering API
  • Personalized product discovery

🛒 Smart Shopping Cart

I built a cart that just works the way you'd expect, implementing the Cart API:

  • Built from scratch to be fast and reliable
  • Price and quantity updates happen instantly
  • Checkout process that's straightforward

How I Built It

🚀 Simple Tools, Great Results

HTML CSS Liquid JavaScript jQuery

I used basic web technologies to build something powerful. You don't always need fancy frameworks to make something that works well.

📲 Keeping It Simple

Instead of relying on tons of apps and plugins, I wrote custom code to get exactly what I wanted.

Only One App Used:

Just a search and filtering app - everything else is custom-built
